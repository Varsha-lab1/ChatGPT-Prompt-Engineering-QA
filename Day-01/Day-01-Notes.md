# Day 1 – Prompt Engineering Fundamentals

## What is Prompt Engineering?

Prompt Engineering is the process of writing clear and structured instructions that help AI generate accurate, relevant, and high-quality responses.
For QA Engineers, prompt engineering improves productivity by helping generate:

- Test Cases
- Bug Reports
- Test Plans
- SQL Queries
- API Test Cases
- Automation Scripts
- Documentation

---
